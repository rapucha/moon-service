Moon path visual exploration

Generic silhouette scale mockups

These variants keep the current Moon pass card and dot-based Moon path, then add a crude foreground scale for low-altitude intuition. The silhouettes are intentionally generic and non-authoritative.

Option A: left silhouette strip

Most direct altitude aid. The plot gives up a narrow left gutter, and silhouettes sit beside the Moon path without covering light buckets.
